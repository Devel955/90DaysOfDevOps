# Day 24 – Advanced Git

## Fast Forward Merge
When the main branch has not moved forward and Git simply moves the pointer to the latest commit of the branch.

## Merge Commit
Git creates a merge commit when both branches have different commits.

## Merge Conflict
Occurs when the same line of a file is changed in two branches.

## Rebase
Rebase moves commits from one branch and replays them on another base branch.

## Merge vs Rebase
Merge keeps history with merge commits.
Rebase creates a linear history.

## Squash Merge
Squash merge combines multiple commits into one single commit.

## Git Stash
Git stash temporarily saves uncommitted changes.

### stash pop vs stash apply
pop → applies and removes stash  
apply → applies but keeps stash

## Cherry Pick
Cherry pick applies a specific commit from one branch to another.

### Risks
Cherry picking may cause conflicts or duplicate commits.
